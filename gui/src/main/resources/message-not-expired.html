This package is based on the latest OpenMarkov core for the Windows platform, with an added plugin for using the DESnets formalism. This plugin is at an early stage of implementation, because of which some of its functionalities are currently limited. However, these limitations still allow evaluating the usage of DESnets, and obtaining accurate simulation results.

The following functionalities are fully usable:

  • Building DESnet networks
  • Setting decision criteria
  • Using a variety of potentials (Function, Increment, Table, Distribution Table, Transition Table, Tree with Events, Exact)
  • Evaluating of a DESnet for a desired experiment size
  • Generating a detailed trace of the evaluation algorithm, stored in ./DESNetFiles/TextualLog
  • Obtaining evaluation results as a table, exportable to CSV
